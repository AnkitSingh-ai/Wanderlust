# Wanderlust

Wanderlust is a web application designed for travelers to explore, book, and manage accommodations and experiences. The platform provides a user-friendly interface and a rich set of features to enhance the travel planning experience.

## Features

- **Search Accommodations**: Easily search for stays based on location, dates, and preferences.
- **Interactive Maps**: Utilize maps to explore properties and experiences in different locations.
- **User Authentication**: Securely sign up, log in, and manage your profile.
- **Property Listings**: View detailed property listings with images, amenities, and reviews.
- **Booking Management**: Manage your reservations and track your bookings.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Technologies Used

- **Frontend**: JavaScript, EJS, CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Other Tools**: Map APIs for interactive maps
